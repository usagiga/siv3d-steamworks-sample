# siv3d-steamworks-sample

Steamworks SDK を Siv3D(siv6) で動かすチュートリアル

## 開発環境

- Visual Studio 2019
    - Siv3D (siv6) August 2016 v2 for VS2019
    - Windows 10 SDK 10.0.17763.0
    - MSVC v140 ビルドツール

VS2015での動作は未保証です。
